# 🏛️ Emulador Hercules

*El motor fundamental que hace posible tu mainframe personal en cualquier computadora*

## 📁 Contenido de esta carpeta

Esta carpeta contiene toda la documentación del emulador Hercules - el software que permite ejecutar sistemas mainframe IBM en computadoras modernas.

---

- **Propósito:** Guía de instalación definitiva
- **Versiones soportadas:** Windows, Linux, macOS
- **Contenido:** 150+ páginas con configuración, red, troubleshooting
- **Ideal para:** ✅ Administradores y usuarios avanzados

### 📖 **HerculesGeneralInfo.pdf** *(3.6 MB)*
- **Propósito:** Conceptos y arquitectura del emulador

- **Para:** ✅ Quienes quieren entender los fundamentos

### 📋 **HerculesUserReference.pdf** *(3.1 MB)**
- **Propósito:** Referencia completa de comandos
- **Contenido:** Todos los comandos, sintaxis, ejemplos

### 📊 **HerculesOperationsandUtilities.pdf** *(1.2 MB)**
