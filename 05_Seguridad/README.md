## 🔐 **¿Qué es RAKF?**

RAKF (Resource Access Control Facility) es un sistema de seguridad completo diseñado específicamente para MVS 3.8J, basado en el probado sistema ESG Security System.

### 🎯 **Funcionalidades Principales:**
- 🔐 **Control de acceso** a recursos del sistema
- 👥 **Gestión de usuarios** y permisos
- 📊 **Auditoría y logging** de actividades
- 🔒 **Criptografía** y seguridad de datos
- 🛡️ **Protección** contra accesos no autorizados

---

## 🚀 **¿Por qué necesitas RAKF?**

En un entorno mainframe, especialmente uno accesible vía emulación, la seguridad es crucial:

- **🏢 Entorno educativo:** Múltiples usuarios necesitan diferentes niveles de acceso
- **🌐 Acceso remoto:** Hercules permite conexión desde cualquier lugar
- **📚 Datos sensibles:** Aunque sea un sistema de aprendizaje, puedes tener información importante
- **🔍 Cumplimiento:** Buenas prácticas de seguridad en todo entorno
